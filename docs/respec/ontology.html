14개 공공 데이터셋과 1개 추론 체인으로 구성된 전국 반려동물 지식그래프의
어휘. UK Public Sector URI 101 (def/id/doc/set 4-layer) 정책을 따른다.

본 명세는 docs/respec/terms.yaml 단일 진실원천에서 자동 생성되며, backend/kg/vocab.py 와의 정합이 빌드 시 강제된다.

1. Namespaces

PrefixIRI용도

2. Datasets (R2RML-style mappings)

14개 공공 데이터셋과 1개 추론 파생 데이터셋의 CSV → RDF 매핑 명세. 각 데이터셋은 backend/kg/builders/dsXX_*.py 정본 빌더에 1:1 대응한다. R2RML(RDB 전용) 대신 CSV 친화적 단순 형식으로 표현하나 rr:logicalTable·rr:subjectMap·rr:predicateObjectMap 의 세 핵심 개념(source, uri template, column → predicate)을 모두 보존한다.

5. Reasoning Patterns

SymptomCategory --def:mapsTo→            VetDepartment
Symptom         --def:indicatesDisease→  Disease    (역방향 def:hasSymptom)
Disease         --def:treatedByDept→     VetDepartment
VetDepartment   --def:treatsSympCategory→ SymptomCategory
VetDepartment   --def:handledBy→         AnimalHospital

예시 SPARQL:

PREFIX def: <https://knowledgemap.kr/koah/def/>
SELECT ?d ?dept WHERE {
  ?sym a def:Symptom ; def:indicatesDisease ?d .
  ?d   def:treatedByDept ?dept .
} LIMIT 10
